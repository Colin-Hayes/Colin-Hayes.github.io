Classifying Galaxies¶

The goal of this project is to automatically classify galaxies using a deep neural network, taking in a picture of a galaxy and deciding what category it falls into (disc, round, cigar shape, etc).

The pictures are from the Sloan Digital Sky Survey and the labels from the Galaxy Zoo. We will then write and train a neural network to do this classification.

Neural Network¶

Next the neural network is going to take the input vector and output a classifcation vector, the 5 different types of galaxies.

The equation for how to do this is $v_{out} = 1.0/ \big( 1+\exp(-(W_2 \textrm{tanh}(W_1 v_{in} + b_1) +b_2))\big)$ where

  • W1 is a size d x 4761 matrix
  • W2 is a size 5 x d matrix
  • b1 is a size d vector
  • b2 is a size 5 vector

where d is the number of neurons in the network. let d =100

The last term: $1/(1+e^{-x})$ ensures that all the numbers in $v_{out}$ are between 0 and 1.

To make this function work in parallel on multiple $v_{in}$'s simultaneously, you can just supply them as a big matrix $V_{in}$ (of size (17457, 4761)) and add in some transpositions to make the matrix multiplication shapes work:

$$ v_{out} = \left(1+\exp\left[-\left(\left[W_2 \cdot \mathrm{tanh}(W_1 \cdot V_{in}^\top + b_1)^\top\right]^\top +b_2\right)^\top\right]\right)^{-1} $$

The output is going to give a value between 0 and 1 for each classification. The index with the greatest number will be the classification that the model predicts. We can make sure that our equation is right by calling the Neural Network, the answer will not be right as the network is not trained yet.

def NeuralNetwork(params,imageVector):
   W1=params[0]
   b1=params[1]
   W2= params[2]
   b2 = params[3]
   h = jnp.tanh(W1 @ imageVector.T + b1).T
   power = -((W2@h).T + b2).T
   output = (1+jnp.exp(power))**(-1)
   return output

Loss¶

Now we need to calculate the Loss to train the nueral network

To quantify the loss we can use cross entropy to calculate the difference between the predicted value of the classification and the actual value. The cross-entropy is defined as

$$ \sum_i -y_i \log(a_i) - (1-y_i)\log(1-a_i)$$

where $y_i$ is the $i'$th component of the exact label and $a_i$ is the $i'$th component of the label you get as output from the neural network.

So now for the image we looked at above we can now calculate the "loss" using the cross entropy:

In [10]:
def loss(params, imageVector, correctLabel):
  y = correctLabel
  a = NeuralNetwork(params,imageVector)
  sum = 0
  for i in range(len(a)):
    sum += -y[i]*jnp.log(a[i])-(1-y[i])*jnp.log(1-a[i])
  return sum
crossentropy = loss(params,train_images[3],train_labels[3])

def fractionCorrect(params,images,labels):
  correct = 0
  for i in range(images.shape[0]):

    #classification vector
    out = NeuralNetwork(params,images[i])

    #the greatest index in the classification vector
    predicted = np.argmax(out)
    actual = np.argmax(labels[i])

    if (predicted == actual):
      correct +=1
  return correct/images.shape[0]

To make an objective function better, step the paramaters of the nework along the gradient. Jax can be used to get the gradient.

The gradient is loss_grad = jit(jax.grad(jit(loss))) and jit helps to speed up the calculation

def fractionSecondCorrect(params,images,labels):
  correct = 0
  counter = 0
  for i in range(images.shape[0]):
    out = NeuralNetwork(params,images[i])
    predicted = np.argmax(out)
    actual = np.argmax(labels[i])
    if (predicted != actual):

Deep Neural Network¶

The neural network above is a shallow network as it only contains 1 hidden layer. To try and develop a better model we can use kerqas to implement a deep neural network.

Using keras we can implement an convolutional neural network to classify the galaxies. A benefit of a convolutional neural network is it preforms translation-invariant learning efficiently which will be beneficial for identifying galaxies that can be oriented in any direction.

Above we altered the image data into a vector, but for the convolutional neural network it will be optimal to keep the data in a 2d array.

def load_data():
    with gzip.open("dataG.pkl.gz", 'rb') as f:
        train_images_raw, train_labels, test_images_raw, test_labels = pickle.load(f)

    #Adds a demention to represent the channel(needed for the CNN) and normalizes the data
    train_images = np.expand_dims(train_images_raw, axis=-1) / 255.0
    test_images = np.expand_dims(test_images_raw, axis=-1) / 255.0

    return train_images, train_labels, test_images, test_labels

building the convolutional neural network¶

The model will be sepereated into 3 Convolutional blocks with the structure:

Conv2D → BatchNorm → Conv2D → MaxPooling

The goal is that each block can interpret different features. With the 32 filter block capturing features such as edges and gradients and as the filters increase, the scale of the features increases. So the 128 layer captures the galaxy shape as a feature. Using a Kernel size (3,3) allows for each point to be put into spatial context and interpret small details.

MaxPooling2D is used to reduce the dimenionality while preserving key features. So now when we apply the next convolitional block, the spatial context of each point is farther reaching in the image.

BatchNormalization stabilizes training by normalizing activations which reduces sensitivity to weight initialization

Then you flatten out the network into a vector and then reduce to the 5d vector output of galaxy classifications.

Finally, using a Keras optimizer to speed up the training time.

In [14]:
def build_model():
    model = models.Sequential([
        layers.Conv2D(32, (3,3), activation='relu', input_shape=(69,69,1)),
        layers.BatchNormalization(),
        layers.Conv2D(32, (3,3), activation='relu'),
        layers.MaxPooling2D((2,2)),

        layers.Conv2D(64, (3,3), activation='relu'),
        layers.BatchNormalization(),
        layers.Conv2D(64, (3,3), activation='relu'),
        layers.MaxPooling2D((2,2)),

        layers.Conv2D(128, (3,3), activation='relu'),
        layers.BatchNormalization(),
        layers.Conv2D(128, (3,3), activation='relu'),
        layers.MaxPooling2D((2,2)),


        layers.Flatten(),
        layers.Dense(128, activation='relu'),
        layers.Dropout(0.3),
        layers.Dense(64, activation='relu'),
        layers.Dense(32, activation='relu'),
        layers.Dense(5, activation='softmax')
    ])

    #Keras Optimizer
    model.compile(optimizer=keras.optimizers.Adam(learning_rate=0.001),
                 loss='categorical_crossentropy',
                 metrics=['accuracy'])
    return model

def train(model, train_images, train_labels):
    #Create a validation set for training
    train_images, val_images, train_labels, val_labels = train_test_split(
        train_images, train_labels, test_size=0.2, random_state=42)

    #Using the validation set, we can make sure not to overfit by stopping early
    callbacks = [
        keras.callbacks.EarlyStopping(patience=5, restore_best_weights=True),
        TqdmCallback(verbose=1)
    ]

    #Training with mini batchs to increase the efficientcy of training the model
    history = model.fit(
        train_images, train_labels,
        batch_size=32,
        epochs=20,
        validation_data=(val_images, val_labels),
        callbacks=callbacks,
        verbose=0
    )
    return model, history
