Fluid Dynamics Simulation

Results of the Simulation:

Building the Simulation

The goal is a 2D fluid simulation in a tunnel.

A 520x180 grid will represent density values for the fluid at each point. Each point will have nine microscopic degrees of freedom. Each degree of freedom will have a value for the density of fluid moving in that direction.

  • $n_k(i,j)$ where $0\leq k \leq 8$ and $(i,j)$ are the indicies over the tunnel
  • $n_0$: stationary fluid moving at velocity $v_0=(0,0)$
  • $n_1$: fluid moving up moving at velocity $v_1=(0,1)$
  • $n_2$: fluid moving down at velocity $v_2=(0,-1)$
  • $n_3$: fluid moving right at velocity $v_3=(1,0)$
  • $n_4$: fluid moving left at velocity $v_4=(-1,0)$
  • $n_5$: fluid moving left-down at velocity $v_5=(-1,-1)$
  • $n_6$: fluid moving left-up at velocity $v_6=(-1,1)$
  • $n_7$: fluid moving right-down at velocity $v_7=(1,-1)$
  • $n_8$: fluid moving right-up at velocity $v_8=(1,1)$

First we will set up the global variables and show the microscopic degrees of freedom:

Given the microscopic densities, there are two macroscopic quanties:

  • the macroscopic density $\rho(i,j)=\sum_k n_k$
  • the macroscopic velocity $\vec{u}(i,j) \equiv (u_x(i,j),u_y(i,j))$ where
    • $u_x(i,j) = 1/\rho \sum_k v_{k,x} n_k(i,j)$
    • $u_y(i,j) = 1/\rho \sum_k v_{k,y} n_k(i,j)$

Now we will create a function to calculate the macroscopic values given the grid(microscopic values):

In[2]:
def Micro2Macro(n):
  n_c =n.copy()

  p = np.zeros((nx, ny))
  for k in range(9):
    p += n_c[k]

  u = np.zeros((2, nx, ny))
  for i in range(n_c.shape[0]):
    u[0] += v[i,0] * n_c[i]
    u[1] += v[i,1] * n_c[i]
  u[0]/=p
  u[1]/=p

  return p, u

Given the macroscopic densities $\rho$ and $\vec{u}$, we need to calculate the equilibrium microscopic densities $n_{eq}$ (size: $9 \times n_x \times n_y$) in order to calculate the collisions between densities and objects later on.

This is a Lattice Boltzman simulation and the equilibrium microscopic densities equation for this model in terms of the previous variables is:

$$n_k^{eq} = \omega_k \rho \left(1 + 3 \vec{v_k} \cdot \vec{u} + \frac{9}{2}(\vec{v_k} \cdot \vec{u})^2- \frac{3}{2}(\vec{u}\cdot \vec{u}) \right) $$

For a 2DQ9 model the omega values are:

  • $\omega_{0}=4/9$
  • $\omega_{1-4}=1/9$
  • $\omega_{5-8}=1/36$
In[3]:
def Macro2Equilibrium(r, u_):
  rho = r.copy()
  u = u_.copy()

  #setting up the omega values
  w= np.array([4/9]+[1/9]*4+[1/36]*4)

  #writing the equation
  neq = np.zeros((9, nx, ny))
  u_dot_u = u[0]**2 + u[1]**2
  for k in range(9):
    v_dot_u = v[k, 0]*u[0] + v[k, 1]*u[1]
    neq[k] = w[k]*rho * (1 + 3 * v_dot_u + 9/2 * (v_dot_u**2) - 3/2 *u_dot_u)

  return neq

Now for the collision step, when microscopic densities collide, the new microscopic densities are

$$n_{post} = n \cdot (1-\omega) + \omega \cdot n_{eq}$$

  • $\omega$ is the relaxation parameter, which is inversly proportional to the viscosity of the fluid. It determins how fast the liquid approaches equilibrium. Let $\omega$ = 1.9572953736654806 to represent airflow.
  • $n_{eq}$ is the equalibrium state that is calculated in the Macro2Equilibrium() function.

After the post collision state is calculated, then the densisties at the position of the obsticle need to remain the same.

In[4]:
def Collision(n, obs):
  rho, u = Micro2Macro(n)
  neq = Macro2Equilibrium(rho, u)
  omega = 1.9572953736654806
  nout = n.copy()*(1-omega) + omega * neq
  for k in range(9):
    nout[k][obs] = n[k][obs]
  return nout

Now the fluid densities need to move in the direction of their velocities. So a function Move(n,obstacle) is needed to carry out this proceedure. The two steps of the function can be broken down into

  • Bounce(n,obstacle)
    • Every velocity where the obstacle is gets reversed
  • MoveDensity(n)
    • Move all your densities in the direction of the velocity
In[5]:
def Bounce(n,obs):
  n_c=n.copy()
  swap_indices = [0,2,1,4,3,8,7,6,5]
  for k in range(9):
    n_c[k,obs] = n[swap_indices[k], obs]
  return n_c

def MoveDensity(n):
  roll = n.copy()
  for k in range(9):
    roll[k] = np.roll(roll[k],v[k],(0,1))
  return roll

def Move(n, obs):
  new_n = Bounce(n, obs)
  roll = MoveDensity(new_n)
  return roll

To implement flow, we need to set boundry conditions to the left and

Left: we are going to assume that there is a flow - this means that the microscopic densities are the same at each step. Therefore, what we should do is simply replace the current densities on the left-most row with the initial microscopic densities (n_init).

Right: we are going to assume that the gradient is zero - i.e. the important physics has dissapeared by this point. To do this, we will set

In[6]:
def FixBoundary(n, n_init):
  Left_vs = [4,5,6]
  n_c = n.copy()
  n_c[:,0,:] = n_init[:,0,:]
  for k in Left_vs:
    n_c[k,-1,:] = n[k,-2,:]
  return n_c

Setup initial conditions:

  • $\rho$ of size $(n_x,n_y)$ (uniformly equal to 1.0)
  • macroscopic velocity $\vec{u}(i,j)$ (sizes $(2,n_x,n_y)$)
    • 0.04*(1.0+1e-4*np.sin(y/(ny)*2*np.pi)) in the x-direction to introduce a small anisotropy to the system.
    • zero in the y-direction
  • compute the equilibrium density $n_{eq}$ associated with the initial density and velocities.
In[7]:
def Setup():
  rho = np.ones((nx,ny))
  u = np.zeros((9, nx, ny))
  y = np.linspace(0, ny, ny)
  u[0] = 0.04*(1.0+1e-4*np.sin(y/(ny)*2*np.pi))
  neq = Macro2Equilibrium(rho, u)
  return neq

Now a function is needed to run the simulation using all the steps implemented above. Run(steps,record,n,n_init,obstacle)

  • the number of steps steps to run
  • how often to record the velocity and density
  • the microscopic densities n to start running from
  • the initial microscopic densities n_init for the boundary conditions
  • the obstacle boolean array.

Returning

  • a list of macroscopic densities rhos
  • a list of macroscopic velocities us
  • the last microscopic density n
In[8]:
def Run(steps,record,n,n_init,obstacle):
  n_c = n.copy()
  rhos = []
  us = []
  for step in range(steps):
    n_c = FixBoundary(n_c, n_init)
    n_c = Collision(n_c, obstacle)
    n_c = Move(n_c, obstacle)
    if (step%record == 0):
      rho, u = Micro2Macro(n_c)
      rhos.append(rho)
      uMag = (u[0]**2+u[1]**2)**(1/2)
      us.append(uMag.T)
  return rhos, us, n_c

from airfoils import Airfoil
def GenerateCylinderObstacle():
  obstacle=np.empty((nx,ny),dtype='bool')
  obstacle[:,:]=False
  x_c = nx / 4
  y_c = ny / 2
  r = ny / 9

  for x in range(nx):
    for y in range(ny):
      if (x - x_c)**2 + (y-y_c)**2 < r**2:
        obstacle[x,y] = True
  return obstacle

def GenerateWallObstacle():
  obstacle=np.empty((nx,ny),dtype='bool')
  obstacle[:,:]=False

  obstacle[50:61, ny//4:3*ny//4] = True
  obstacle[200:210, ny//4:3*ny//4] = True
  return obstacle

def GenerateAirfoil():
  foil = Airfoil.NACA4('4812')
  obstacle = np.zeros((nx, ny), dtype=bool)
  airfoil_chord_length = nx//4
  start_x = 100
  x_coords = np.linspace(0, 1, airfoil_chord_length)
  y_upper = foil.y_upper(x_coords) * (ny//1.2)
  y_lower = foil.y_lower(x_coords) * (ny//1.2)
  for i, x in enumerate(x_coords):
      grid_x = int(start_x + i)
      grid_y_upper = int(ny // 2 + y_upper[i])
      grid_y_lower = int(ny // 2 + y_lower[i])
      obstacle[grid_x, grid_y_lower:grid_y_upper + 1] = True
  obstacle = np.fliplr(obstacle)
  return obstacle

def Animate(us_flat,vMax):
    fig, ax = plt.subplots()
    cax = ax.imshow(us_flat[1],cmap=plt.cm.Reds,vmin=0,vmax=vMax)
    plt.close(fig)
    def animate(i):
         cax.set_array(us_flat[i])

    anim = FuncAnimation(fig, animate, interval=100, frames=len(us_flat))
    return anim
